اليوم التدريبي

كل يوم تدريبي هو وحدة تعليمية متكاملة. المعلم يُصممها بنشاطات متنوعة تغطي جميع جوانب اللغة.

  • ✓ أنشطة متنوعة: مفردات، قواعد، استماع، محادثة، قراءة، كتابة
  • ✓ مواعيد نهائية إرشادية (لا تمنع المشاركة)
  • ✓ حد أدنى اختياري للأنشطة المطلوبة
  • ✓ ترتيب تلقائي — لا حاجة للتنسيق اليدوي
  • ✓ إعلانات المعلم تظهر مباشرة للطلاب
